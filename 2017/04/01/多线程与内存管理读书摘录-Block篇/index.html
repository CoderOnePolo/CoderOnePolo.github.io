多线程与内存管理读书摘录(Block篇)

什么是Blocks

Blocks:带有自动变量（局部变量）的匿名函数，匿名函数就是不带有名称的函数。

Block语法

1.没有函数名

2.带有“^”

^ 返回值类型 ``参数列表

const char text[] = "hello";
void (^blk2)(void ) = ^{
    text[2];
};
//error:cannot refer to declaration with an array type inside block
修改
   const char *text = "hello";

int main()
{
void (^blk)(void) = ^{printf(“Block\n”);};
blk();
return 0;
}
